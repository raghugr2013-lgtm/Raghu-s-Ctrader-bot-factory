## INTEGRATION WITH BOT FACTORY

After conversion, import CSV data into Bot Factory:

### Method 1: Direct API Upload
Use the market data import endpoint to upload CSV programmatically.

### Method 2: Manual Database Import
```python
from pymongo import MongoClient
import pandas as pd
from datetime import timezone

# Read CSV
df = pd.read_csv('eurusd_h1.csv')

# Connect to MongoDB
client = MongoClient('mongodb://localhost:27017')
db = client['trading_db']

# Import candles
for _, row in df.iterrows():
    candle = {
        'symbol': 'EURUSD',
        'timeframe': '1h',
        'timestamp': pd.to_datetime(row['time']).replace(tzinfo=timezone.utc),
        'open': float(row['open']),
        'high': float(row['high']),
        'low': float(row['low']),
        'close': float(row['close']),
        'volume': float(row['volume']),
        'provider': 'dukascopy',
    }
    db.market_candles.update_one(
        {
            'symbol': candle['symbol'],
            'timeframe': candle['timeframe'],
            'timestamp': candle['timestamp']
        },
        {'$set': candle},
        upsert=True
    )
```

## TROUBLESHOOTING

### Common Issues:

❌ "No .bi5 files found"
   → Check input folder structure matches YYYY/MM/DD/HH format

❌ "Failed to decompress"
   → File may be corrupted, re-download from Dukascopy

❌ "Point value seems incorrect"
   → Use --point flag to manually specify

❌ "Too many NaN values"
   → Source data has gaps, converter will forward-fill
